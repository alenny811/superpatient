Upgrade instructions
====================

* Backup everything as usual (code and db)

* Apply the database update script `db/update_db.v4.27.sql`


New features
============

* Track the database version and complain if it doesn't match what the code expect

* Display the site associated with a patient or its consultations

* Support multiple sites sharing a single database (was badly overlooked)

* Added indexes on the sites columns of the patients and consultations

Bug Fixes
=========

* Fix bills always using the application site instead of the bill's one

* Fix cancelation warning on bills talking about the consultation while it actually
  concerned only the bill

* Fix bills created at a later time instead of the consultation time not using the
  consultation site
